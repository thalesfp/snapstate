# @snapstore/form

Zod-powered form stores for [Snapstore](https://github.com/thalesfp/snapstore): validation, DOM binding, and a submit lifecycle.

```bash
npm install @snapstore/form
```

Requires `react >= 18` and `zod >= 4`. See the [full documentation](https://github.com/thalesfp/snapstore/blob/main/docs/forms.md).
