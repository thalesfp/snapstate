# @snapstore/react

React integration for [Snapstore](https://github.com/thalesfp/snapstore): `SnapStore` with `connect()` and `SnapStore.scoped()`.

```bash
npm install @snapstore/react
```

Requires `react >= 18`. See the [full documentation](https://github.com/thalesfp/snapstore/blob/main/docs/react-integration.md).
